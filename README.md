# Prueba-tecnica
Proyecto de una aplicación web para gestionar tareas (crear, editar y eliminar) usando:
PHP con arquitectura MVC

* AJAX / jQuery para operaciones sin recargar la página

* MySQL como base de datos

* HTML + CSS

# Instalacion
<p>1 - Clonar el proyecto en un servidor local o remoto</p>
<p>2 - Ejecute en su servidor MySQL el script tasks.sql de la base de datos que esta en la carpeta DB</p>
<p>3 - Configure el archivo server.php con los datos de su servidor MySQL. El archivo esta en la carpeta config</p>
<p>4 - Servidor por defecto</p>
<p>DB_SERVER="localhost"</p>
<p>DB_USER="root"</p>
<p>DB_PASS=""</p>
